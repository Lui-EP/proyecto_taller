# Proyecto Taller - MercadoLocal

Proyecto completo con:

- `backend-micro/` -> backend principal (FastAPI, 2 microservicios)
- `frontend-react/` -> app web (React + Vite)
- `frontend-native/` -> app movil (React Native + Expo)
- `backend/` -> backend legacy (Node, solo referencia)

## Orden recomendado para levantar todo

1. Levantar backend micro (`backend-micro/README.md`)
2. Levantar web (`frontend-react/README.md`)
3. Levantar movil (`frontend-native/README.md`)

## Puertos clave

- Clientes service: `8001`
